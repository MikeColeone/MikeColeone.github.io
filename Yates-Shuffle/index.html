Yates Shuffle

随机洗牌算法

  1. 基本思想：从原数组取从未渠道的数字到新数组

  2. 步骤

    1. 初始化原始数组新数组 n

    2. 从还没处理的数组（k）随机产生一个[0,k)之间的数字p

    3. 从剩下的k个数字中去除第p个数字

    4. 重复步骤

  3. 随机性验证：

    1. 第一步：从 n 个元素中选一个进行交换，每个元素被选中的概率是 1/n。

    2. 第二步：从 n-1 个元素中选一个进行交换，每个元素被选中的概率是 1/(n-1)。

    3. 依此类推，直到最后剩下的一个元素。

    最终，每个元素出现在数组任意一个位置的概率是：1/n!

    发现是分母是全排列的次数
