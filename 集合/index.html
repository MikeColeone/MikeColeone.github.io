map

在 Java 中，Map 是一个接口，常见的实现类包括 HashMap、TreeMap、LinkedHashMap 等。Map 提供了一系列操作键值对的方法，以下是 Map 常用的函数（方法）及示例代码。

1. 基本操作方法

1.1 put(K key, V value)

  • 功能：向 Map 中添加一个键值对。
  • 说明：如果键已存在，新值会覆盖旧值。

1.2 get(Object key)

  • 功能：根据键获取对应的值。
  • 说明：如果键不存在，返回 null。

1.3 containsKey(Object key)

  • 功能：检查 Map 中是否包含指定的键。
  • 返回值：true 或 false。

1.4 containsValue(Object value)

  • 功能：检查 Map 中是否包含指定的值。

1.5 remove(Object key)

  • 功能：根据键删除对应的键值对。
  • 返回值：被删除键对应的值，如果键不存在，返回 null。

1.6 size()

  • 功能：返回 Map 中键值对的数量。

1.7 isEmpty()

  • 功能：检查 Map 是否为空。

1.8 clear()

  • 功能：清空 Map 中的所有键值对。

2. 批量操作方法

2.1 putAll(Map<? extends K, ? extends V> m)

  • 功能：将另一个 Map 中的所有键值对复制到当前 Map 中。

3. 遍历 Map 的方法

3.1 keySet()

  • 功能：返回 Map 中所有键的集合。

3.2 values()

  • 功能：返回 Map 中所有值的集合。

3.3 entrySet()

  • 功能：返回 Map 中所有键值对（Map.Entry）的集合。

3.4 使用 Lambda 表达式遍历（Java 8 及以上）

4. 其他方法

4.1 replace(K key, V value)

  • 功能：替换指定键的值，如果键存在。

4.2 replace(K key, V oldValue, V newValue)

  • 功能：只有当旧值匹配时，才会替换为新值。

4.3 getOrDefault(Object key, V defaultValue)

  • 功能：获取指定键的值，如果键不存在，则返回默认值。

5. 特殊方法：与 Java 8 Stream 结合

5.1 使用 stream() 对 Map 进行过滤

  • 功能：使用 Stream 流处理 Map 数据。

总结：Map 常用方法列表

方法 功能
put(K, V) 添加或替换键值对
get(Object key) 根据键获取值
containsKey(Object key) 判断是否存在指定键
containsValue(Object v) 判断是否存在指定值
remove(Object key) 根据键删除键值对
size() 返回键值对数量
isEmpty() 判断 Map 是否为空
clear() 清空 Map
putAll(Map m) 将另一个 Map 数据复制过来
keySet() 返回所有键的集合
values() 返回所有值的集合
entrySet() 返回键值对的集合
replace(K, V) 替换指定键的值
replace(K, V, V) 条件替换值
getOrDefault(K, V) 获取值，若无返回默认值

常见实现类

  1. **HashMap**：无序存储，允许 null 键和值。
  2. **TreeMap**：键有序存储（基于自然排序或比较器）。
  3. **LinkedHashMap**：保持插入顺序。
  4. **ConcurrentHashMap**：线程安全的 HashMap。

这些方法在不同 Map 实现类中都可以使用，选择合适的 Map 类型才能更好地满足业务需求。
